# App-Shortcuts-Example
### 목적

Android Dynamic ShortCuts 기능을 구현한다.



### 요구사항
##### 메인페이지

1. Github Id를 저장 한다.
2. 저장한 Github Id 들은 리스트 형태로 표현된다.
3. 저장한 Github Id 들은 shortcut에 추가된다.

##### 프로필페이지

1. Github Api를 통해 데이터를 가져온다.

2. 프로필에 대한 정보를 표시한다.

   

#### Shortcut
1. 저장된 프로필의 이름을 표시한다.

2. 선택시 프로필페이지로 이동한다.

   

#### 기술

* Kotlin

* Shortcuts

* Coroutine
